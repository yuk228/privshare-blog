# Privshare
Privshareは、プライベシーや匿名化に関するコアな情報を提供する、オープンソースのプログサービスです。

私たちの生活に欠かせないインターネット上で、個人データやオンラインでの行動情報を販売する企業や、検閲・監視を行う当局から逃れましょう。
